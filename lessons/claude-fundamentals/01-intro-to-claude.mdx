# Introduction to Claude

Claude is a family of AI assistants built by **Anthropic**, designed to be helpful, harmless, and honest.

## What is Claude?

Claude is a large language model (LLM) that can:
- Answer questions and explain concepts
- Write and debug code
- Analyze documents and data
- Build complex multi-step workflows with tools

## Key Concepts

### Models

Anthropic provides several Claude models with different capability/cost tradeoffs:

| Model | Description | Best For |
|---|---|---|
| claude-opus-4 | Most capable | Complex reasoning, agents |
| claude-sonnet-4 | Balanced | Most production tasks |
| claude-haiku-3 | Fastest/cheapest | High-volume, simple tasks |

### Messages API

The core of every Claude integration is the **Messages API**:

```python
import anthropic

client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "What is the capital of France?"}
    ]
)

print(message.content[0].text)
# → "The capital of France is Paris."
```

### Tokens

Claude processes text as **tokens** — roughly 4 characters per token or ¾ of a word. Understanding tokens is important for:
- **Cost estimation** — you pay per input/output token
- **Context limits** — models have a maximum context window
- **Performance** — longer prompts are slower and more expensive
